このページは開発確認用です。入力した Basic 認証情報は、このブラウザ上の API 呼び出しにだけ使います。

認証

メモグリッド作成

MapArea の範囲は、中心座標・1マスの大きさ・縦横のマス数から自動計算されます。 north / south / east / west は作成後にサーバー側で計算されます。

一般ユーザーは最大500マス、南北30,000m、東西30,000mまで作成できます。

メモグリッド

自分が作成したメモグリッドは一覧から削除できます。共有メモグリッドは削除できません。

まだ読み込んでいません。

GridCell

メモグリッドを選択すると、自動生成済みの GridCell を表示します。

先にメモグリッドを選択してください。

共有相手はまだ読み込んでいません。
